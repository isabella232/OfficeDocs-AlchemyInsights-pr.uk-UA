# <a name="activate-microsoft-forms-for-an-organization"></a>Активація Microsoft Forms для організації

Адміністратор може вмикати та вимикати Microsoft Forms для своєї організації. Дізнайтеся, як перевірити активацію ліцензій і стан [Azure CollabDBService](https://support.office.com/article/Turn-off-or-turn-on-Microsoft-Forms-8dcbf3ab-f2d6-459a-b8be-8d9892132a43) для вашого клієнта.
