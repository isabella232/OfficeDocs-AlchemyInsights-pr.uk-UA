# <a name="microsoft-defender-for-office-365-setup-guide"></a>Посібник із налаштування Microsoft Defender для Office 365

Посібник із настроювання [Захисника Microsoft для Office 365](https://go.microsoft.com/fwlink/?linkid=2146614) допомагає захистити організацію від зловмисних загроз, які можуть виникнути у вашому середовищі через повідомлення електронної пошти, посилання та засоби співпраці сторонніх постачальників. У цьому посібнику наведено ресурси та відомості, які допоможуть вам підготувати та визначити план Microsoft Defender відповідно до потреб вашої організації.
