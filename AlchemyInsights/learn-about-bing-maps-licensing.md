# <a name="learn-about-bing-maps-licensing"></a>Відомості про ліцензування карт Bing

Служба "карти Bing" пропонує кілька гнучких параметрів ліцензування, які підтримуватимуть потреби та технічні вимоги організації. Крім того, за допомогою вільного ярусу можна почати створювати карти Bing сьогодні.

Докладні відомості наведено в статті [ліцензування служби "карти Bing](https://go.microsoft.com/fwlink/?linkid=2150203)".
