# <a name="limit-sharing-in-microsoft-365"></a>Обмеження спільного доступу в Microsoft 365

Хоча вимкнути внутрішній спільний доступ повністю або видалити кнопку Спільний доступ із сайтів не можна, ви можете обмежити спільний доступ у Microsoft 365 відповідно до потреб організації. Докладні відомості див. в [статті Обмеження спільного доступу в Microsoft 365.](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)
