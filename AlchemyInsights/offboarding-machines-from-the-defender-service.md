# <a name="offboarding-machines-from-the-microsoft-defender-atp-service"></a>Системи від'єднання від АТС Microsoft Defender

Якщо вам потрібна допомога з від'єднання комп'ютера від служби РЗЗ Захисника, див. інструкції для вашої операційної системи:  

- Відомості Windows 10 див. в статті [Від'єднувачі зі служби АТФ Microsoft Defender.](/windows/security/threat-protection/microsoft-defender-atp/offboard-machines#offboard-windows-10-devices)
- Відомості про пристрої, не Windows див. в [цьому Windows.](/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints-non-windows#offboard-non-windows-devices)
