# <a name="office-cloud-policy-service-for-microsoft-365-apps-for-enterprise"></a>Хмарна служба політики Office для програм Microsoft 365 для підприємств

Служба хмарних політик Office (OCPS) дає змогу примусити настройки політики для програм Microsoft 365 для підприємств на пристрої користувача, навіть якщо пристрій не приєднано до домену або іншим способом керується ними. 

Докладні відомості:

[Огляд хмарної служби політики Office для програм Microsoft 365 для підприємств](https://docs.microsoft.com/deployoffice/overview-office-cloud-policy-service) 
 [Огляд консультанта з політики безпеки для програм Microsoft 365 для підприємств](https://docs.microsoft.com/deployoffice/overview-of-security-policy-advisor)
