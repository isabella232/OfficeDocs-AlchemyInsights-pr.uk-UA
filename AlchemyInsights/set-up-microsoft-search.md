# <a name="set-up-microsoft-search"></a>Налаштування пошуку Microsoft

Microsoft Search‎ provides a user-friendly interface to search and find results in your organization across ‎Microsoft 365‎ services and applications to help you complete what you're working on. It is included at no additional cost with your eligible ‎Microsoft 365‎ subscription.

Скористайтеся [посібником із](https://go.microsoft.com/fwlink/?linkid=2156919) настроювання пошуку Microsoft, щоб настроїти пілотний проект для групи користувачів або розгорнути його для всіх користувачів в організації. Призначте адміністраторам пошуку й редакторам пошуку, а потім налаштовуватимете пошук для користувачів, використовуючи відповіді та додаткові параметри, наприклад додамо розширення Bing до Chrome або як стандартну пошукову систему Bing.
