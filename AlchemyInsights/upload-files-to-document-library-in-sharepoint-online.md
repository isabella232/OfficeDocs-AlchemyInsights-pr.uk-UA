# <a name="upload-files-to-onedrive-or-sharepoint"></a>Upload файли для OneDrive або SharePoint

Файли можна передати до бібліотеки документів у SharePoint Online, просто перетягнувши їх із комп'ютера до бібліотеки документів. Завдяки OneDrive можна синхронізувати файли між комп'ютером і хмарою, щоб мати доступ до своїх файлів звідусіль. Синхронізовані файли можна також працювати безпосередньо у Файловому провіднику та відкривати їх навіть без мережі. Докладні відомості про передавання та синхронізацію див. за посиланнями нижче.

- [Upload папки або файли в бібліотеці](https://support.office.com/article/upload-a-folder-or-files-to-a-document-library-eb18fcba-c953-4d45-8d90-8da66edeacdb)

- [Upload за допомогою синхронізації](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-in-windows-615391c4-2bd3-4aae-a42a-858262e42a49)

- [Вирішення проблем із синхронізацією OneDrive](https://support.office.com/article/Fix-OneDrive-sync-problems-83ab0d8a-8400-45b0-8dcf-dc8aa8a6bcf8)
