# <a name="using-custom-policy-in-intune"></a>Використання настроюваної політики в Intune

У Intune є багато параметрів, вбудованих в Android, iOS, macOS Windows пристроях. Ви також можете створити спеціальний профіль, а потім розгорнути ці настроювані параметри на своїх пристроях за допомогою Intune. Настроювані профілі призначено для додавання параметрів і функцій пристрою, не вбудованих у Intune.

[Користувацькі параметри для пристроїв з Android у Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Користувацькі параметри для пристроїв з Android Enterprise у Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Настроювані параметри для пристроїв iOS у Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Користувацькі параметри для пристроїв macOS в Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Користувацькі параметри Windows 10 пристроях в Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Користувацькі параметри Windows Holographic for Business пристроях в Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
